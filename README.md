# m-parlay-guide

## 项目简介

本项目是一个 HTML 页面归档仓库，用于存放和发布一系列独立的网页文档。这些页面内容相互独立，不依附于特定网站或域名，可作为静态资源直接使用或部署。

## 目录说明

```
m-parlay-guide/
├── index.html          # 仓库首页 / 导航页
├── pages/              # 存放主要 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 共用资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：核心页面归档目录，每个 `.html` 文件为一个独立页面。
- `assets/`：页面引用的通用静态资源，按类型分目录存放。

## 页面归档说明

- 所有页面均为纯前端 HTML，无需后端服务即可直接运行。
- 页面之间无功能依赖，可单独打开或部署。
- 页面内容保持原始文件结构，不经过混淆或压缩，方便审阅和修改。
- 归档内容会不定期更新，新增或调整部分页面。

## 维护说明

- 欢迎提交 Issue 报告页面显示问题或提出改进建议。
- 如需贡献新页面，请将 HTML 文件放入 `pages/` 目录，并在 `index.html` 中添加导航链接。
- 请勿在页面中加入外部不可控资源（如第三方 CDN 脚本），以保持归档的长期可用性。
- 本仓库不承担页面具体内容的准确性或时效性责任，请使用者自行判断。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/m-parlay-guide.git
   ```
2. 直接打开 `index.html` 或 `pages/` 下的任意页面即可浏览。

或通过 GitHub Pages 等静态托管服务部署整个仓库，即可在线访问。
